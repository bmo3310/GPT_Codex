#匯率簡易查詢與匯兌差異比較

讓你在兌換幣別時決定要不要再等更好的匯率或者累積更大的金額

## 功能特色

- 美元、台幣、日幣與BTC走勢匯率查詢
- 匯兌金額在不同匯率下的比較
